124 // Solo quando il cronometro è in funzione, eseguo le operazioni di determinazione
125 /*
126 - Ottengo il valore corrente dell'ora espresso in secondi con il metodo "get_EpochTime"
127 - Sottraggo il valore di base 'm_BaseTime' calcolato precedentemente
128 - Aggiungo un eventuale tempo rimasto salvato in memoria nell'attributo 'm_OldTime'
129 - Salvo il tutto nell'attributo 'm_Time'. Il valore sarà espresso in secondi
